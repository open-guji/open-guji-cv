# s5: 拆分半页 (split)

## 概述

| 项目 | 说明 |
|------|------|
| 编号 | s5 |
| 文件夹 | `s5_split/` |
| 源文件 | `guji_preprocess/preprocessors/split_page.py` |
| 类名 | `SplitPagePreprocessor` |
| 执行条件 | `profile.is_uncut == True`（未剪切的筒子页） |
| 输入 | s4 输出（校正后的图片） |
| 输出 | 拆分后的左右两半页 |

## 背景

筒子页（uncut_full）是古籍常见装帧方式：一张纸对折，对折线在外侧（书口）。扫描时如果未剪切，一张图包含左右两个半页。需要沿版心中线拆分。

## 输出格式

返回 `list[tuple[str, np.ndarray]]`：

```python
[("right", 右半页), ("left", 左半页)]
```

Pipeline 会自动生成文件：`{stem}_right.png`, `{stem}_left.png`

古籍从右往左读，右半页在前（奇数页），左半页在后（偶数页）。

## 中线检测算法

```
1. 灰度化
2. 垂直投影：每列的平均亮度
3. 在中央 30% 区域（35%-65%）搜索
4. 滑动窗口平滑（窗口 = 图宽 / 100）
5. 找最暗位置 = 版心中线
```

版心界栏是一条从上到下的暗线，在垂直投影中表现为明显的暗谷。

## 参数

无可调参数。搜索范围和平滑窗口自适应图像尺寸。

## 适用书籍

| 书籍 | 执行 | 原因 |
|------|------|------|
| book2 | 是 | uncut_full，10张→20张 |
| book1/3/4/5 | 跳过 | cut_half，已剪切 |
