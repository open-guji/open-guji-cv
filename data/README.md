## 页面类型说明

- **已剪切半页** (`cut_half`): 每张图片是一个半页，奇数为右半页，偶数为左半页
- **未剪切筒子页** (`uncut_full`): 每张图片是完整筒子页，需从中缝分页
- **对开** (`spread`): 两页平摊拍照，中间有中缝，需分页
- **表格** (`table`): 含表格的特殊页面，不按常规行列排版
